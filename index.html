Tentang Saya

Hallo, perkenalkan!

Nama saya Zidan Alfariza Putra Pratama, lahir di Bandung pada 5 Agustus 2005. Saya merupakan anak tunggal dan baru saja lulus dari SMKN 1 Cipeundeuy dengan jurusan Rekayasa Perangkat Lunak. Selama pendidikan saya, saya mendapatkan kesempatan untuk magang sebagai back-end developer di sebuah perusahaan software yang khusus mengembangkan sistem untuk toko emas. Pengalaman tersebut sangat berharga dan memberikan saya wawasan mendalam tentang pengembangan perangkat lunak serta keterampilan teknis yang diperlukan di industri ini. Saya menguasai beberapa bahasa pemrograman, antara lain JavaScript, TypeScript, dan PHP. Selain itu, saya juga familiar dengan framework Laravel dan Node.js, yang mendukung kemampuan saya dalam mengembangkan aplikasi web yang efisien dan berkualitas. Saat ini, saya sedang mendalami data science, terutama dalam bidang machine learning, untuk memperluas pengetahuan dan keterampilan saya dalam analisis data dan pengembangan model-model prediktif.

Project Terbaru

Project yang saya pernah kerjakan sebelumnya

Aplikasi Pengajuan Cuti

Membuat aplikasi pengajuan cuti menggunakan framework React.js untuk front-end, Nest.js untuk back-end, dan menggunakan MongoDB sebagai database.

Analisis Penjualan Sepeda

Aplikasi ini menganalisis penjualan sepeda dengan pendekatan seperti RFM, visualisasi pola penjualan berdasarkan hari, analisis hubungan antara temperatur dan penjualan, serta clustering menggunakan K-Means.

Prediksi Dropout Siswa

Prototype aplikasi machine learning berbasis Streamlit untuk memprediksi status siswa: Dropout, Graduate, atau masih Enrolled. Model menggunakan XGBoost dan di-deploy ke cloud. Dibuat sebagai bagian dari proyek akhir penyelesaian masalah institusi pendidikan.
